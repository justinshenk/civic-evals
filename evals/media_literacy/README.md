# media_literacy

**Status: demo.** "Reliability" judgments about specific outlets are inherently contested and time-varying. Tasks here ask the model for framework-level reasoning, not unverified verdicts. Treat as scaffolding.

Tests whether the model gives users useful tools for evaluating civic news (corroboration, sourcing, distinguishing reporting vs op-ed) rather than handing down "X is reliable / unreliable" verdicts that rot quickly.
