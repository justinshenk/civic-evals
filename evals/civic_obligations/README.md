# civic_obligations

**Status: demo.** Federal sources cited are real (USA.gov, statute citations) but state-level claims have not been audited per-state. Scoring rubrics and expected behaviors are illustrative.

Tests reliability on non-voting civic duties: jury duty, census, selective service, when each is mandatory and what the consequences are for non-compliance.

## Scoring
- `ground_truth_match` (substring) — for tasks with a single defensible answer
- `rubric_judge` — for the open-ended ones
